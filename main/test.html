my projects

Personal Blog: Eudaemonia

Eudaemonia began as a project for my high school senior year English class. Built from scratch with HTML & CSS, it is a home for the essays I write about the many things I am personally interested in. Hosted on Neocities since April 2023, it has amassed over 10k views and 17 followers.

GeneticGuess Sentencer

Made with pure Python, GeneticGuess Sentencer is a sentence guesser game that attempts to generate a user's sentence randomly as quickly as possible using a genetic algorithm model.

Yu-Gi-Oh! Card Data Analyzer

Made with pure Python, this Yu-Gi-Oh! Card Data Analyzer reads a given CSV file, and displays information based on the user's request. It also has a search function, and can display decklists based on a YDK file

Crossword Puzzle

Made with pure Python, this is a fully functional Crossword puzzle game that reads a given CSV file and turns it into a playable Crossword. It can also display clues, reveal answers, and give hints by revealing a letter at a specified coordinate.

Mortgage Planning Calculator

Made with pure Python, this mortgage planning calculator prompts for information, and calculates monthly payment costs and determines if it is affordable to the user. It can also can print a monthly payment schedule (amortization table), and accounts for multiple cases if certain information is unknown.

Social Media Triangle Relationship Finder

Made with pure Python, this triangle relationship finder takes, reads, and organizes a CSV name file, TXT friend id file, and TXT friend file into a data structure (nested dictionary). It takes multiple user options to find and display information about the relationships and patterns between friend lists.
